日语生词短语语法

日语生词短语语法

真题

2024

完形填空

  1. 手短に言えば

    简而言之

    手短に言えば、彼は仕事を辞めたんです

  2. ひっきりなしに

    不断地，接连不断地

    雨がひっきりなしに降り続いている

    電話がひっきりなしに鳴っている

  3. はず和べき的区别

    はず：语气较为温和，表示预期或逻辑上的必然性。

    べき：语气较为强烈，表示义务、责任或强烈建议。

  4. エッセンス（essence）

    本质、精华

    この本のエッセンスを理解することが重要です

  5. 転がる

    滚动、滚落、摆着

    石が坂道を転がっていく

    その辺には本がたくさん転がっている

阅读1

  1. 遊興性（ゆうきょうせい）

    娱乐性质、消遣倾向

    遊興性のある場所でリラックスしたい

  2. みすぼらしい

    破旧的，寒酸的，邋遢的，不体面的

    彼はみすぼらしい服を着ていた

  3. 台頭する（たいとうする）

    崛起，兴起，显现

    新しい技術が台頭してきている

  4. 見て取る

    看出、察觉、理解

    彼の表情から緊張していることを見て取ることができた

    その場の雰囲気を見て取り、すぐに行動した

  5. 担い手（にないて）

    挑东西的人;中坚;肩负重任的人

    一家の担い手

  6. における和において的区别

    における：表示“在…方面”或“在…的情况下”

    において：表示“在…的场所、领域或情况下”

    教育における問題

    会議において

阅读2

  1. 原発（げんぱつ）

    原子力発電所的缩写

    福島の原発事故は世界的に注目を集めた

  2. 直後（ちょくご）

    正后方，刚…不久

    地震の直後に余震が続いた

  3. 萎縮（いしゅく）

    経済の不況により、多くの企業が萎縮している

  4. ハードディスク

    hard disk,硬盘

  5. 滅多に（めったに）

    很少，不常，几乎不，与否定形式动词搭配

    彼は滅多に映画を見に行かない

  6. 壊滅的（かいめつてき）

    壊滅的な地震が都市を襲った

阅读3

  1. 俗説(ぞくせつ)

    俗说，民间传说

    その病気について多くの俗説がある

  2. くしゃくしゃ

    皱巴巴，揉成一团

    紙をくしゃくしゃにして捨てた

  3. 利口

    聪明，伶俐，机灵

    利口な子供

  4. 驚異的(きょういてき)

    惊人的，可惊的，令人瞠目的

    驚異的な記録

  5. 欠如(けつじょ)

    缺乏，缺少

    責任感が欠如している。

  6. 芸当(げいとう)

    绝技，把戏

    彼女のダンスの芸当はプロ並みだ

阅读4

  1. 躊躇(ちゅうちょ)

    躊躇することなく引き受けた

  2. どんぶり鉢(どんぶりばち)

    丼ものや麺類などを盛り付ける際に使う食器で、プラスチック製や陶器製、木製などがあります

  3. 一年坊主（いちねんぼうず）

    形容那些在某件事情上刚开始时非常热心，但很快就失去兴趣并放弃的人

    新しい趣味を始めても、一年坊主にならないように気をつけている

  4. 多分に甘美で、どこかに秘密にしなければならぬような影を持った物哀しさがあった

    描述某种情感或氛围，特别是带有复杂感情的情景

    过于的甜蜜，带有某种必须隐瞒的阴影的忧伤感

翻译

  1. 呼び寄せる

    召唤、叫来

    鳥を呼び寄せる笛の音

  2. 孕む（はらむ）

    怀有，包含

    危険性をはらんでいます

  3. 図式（ずしき）

    图表、示意图

    この本には多くの図式が含まれています

2023

完型

  1. すがすがしい

    神清气爽的，清新舒畅的

    すがすがしい朝の空気

  2. 口ずさむ

    小声哼唱，小声哼诵诗歌等

    唐詩を口ずさみながら散歩している

  3. 首を傾げる（くびをかしげる）

    歪着头想，纳闷，觉得奇怪

    彼はどうしてまだ電話してくれないの、首を傾げた

  4. 仕草（しぐさ）

    动作，举止，态度

    あいつの仕草が気に入らない

  5. 治療にあたらせる

    让（某人）进行治疗　或　安排（某人）进行治疗

阅读1

  1. 水掛け論

    抬杠，没有休止的争论

    水掛け論を続けるよりも、解決策を見つけることが重要だ

  2. 揚げ足取り

    揭短，找茬

    彼はいつも人の揚げ足取りばかりしている

阅读2

  1. かたじけない（片付けない）

    通常用于对对方的恩惠表示深深的感激

    このような親切をしていただき、かたじけない

  2. 節句（せっく）

    传统节日

  3. なにもかも

    一切、所有的事物

阅读3

  1. 狩猟(しゅりょう)

    狩猟に出かける

  2. 獲物(えもの)

    猎物

    獲物を追って山野を駆けめぐる

  3. 営み(いとなみ)

    行为，工作，事情，准备

    冬の営みをいそぐ

  4. 露地（ろじ）

    大地，露天的地面

    露地栽培

  5. 呻き（うめき）

    呻吟，哼哼

    呻き声

  6. 病い（やまい）

    心の病いは見えにくい

  7. 振り絞る

    竭尽全力

    彼女は勇気を振り絞って、告白した

  8. 抗う（あらがう）

    争；反抗

    あらがいがたい運命

  9. 渾身（こんしん）

    浑身，全身

    渾身の力

阅读4

  1. 分岐点（ぶんきてん）

  2. ハーモニカ

    harmonica 口琴

  3. 天辺（てっぺん）

  4. 差し引く

    减去

  5. やけに

    非常

    やけに寒い

  6. 手付き

    动作、手法

    器用な手付きで箸を使う

  7. 煮しめ

    红烧

  8. 物色（ぶっしょく）

  9. 胡坐（あぐら）

    盘腿坐

  10. 頬張る

    大口吃；把嘴塞满

    お菓子を頬張る

  11. うっとり

    因神魂颠倒而发呆状

    うっとり（と）見とれる

  12. 午睡（ごすい）

翻译

  1. それからまた

    然后又、另外

  2. 打ち砕く（うちくだく）

    打烂，粉碎

    頭を打ち砕かれて死ぬ

  3. 突き抜ける（つきぬける）

    穿透，扎透

    天井を突き抜ける

2022

完型

  1. 見当（けんとう）

    方位，方向，估计

    寺はこの見当にある

  2. 賽（さい）

    骰子

    賽を投げる

  3. 機縁（きえん）

    机缘

    これを機縁に今後もよろしく

  4. 違算（いさん）

    算错;误算;计划错误;估计错

    違算を生ずる

  5. 気の向く

    随心所欲

    今日は気の向くままに過ごそう

  6. うっかり

    不注意，不留神。无意中，稀里糊涂，漫不经心。

    うっかりして乗り越す

  7. 行李（こうり）

    行李，装衣的箱笼

阅读1

  1. とりわけ

    特别，尤其

    彼女はどの料理も好きだが、とりわけ寿司が大好きだ

  2. 手っ取り早い

    迅速的、简单的、快捷的

    手っ取り早く終わらせたいなら、この方法がいい

  3. 紙片（しへん）

    紙片が散らばっている
