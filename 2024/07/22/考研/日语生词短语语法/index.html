日语生词短语语法

真题

2024

完形填空

  1. 手短に言えば

    简而言之

    手短に言えば、彼は仕事を辞めたんです

  2. ひっきりなしに

    不断地，接连不断地

    雨がひっきりなしに降り続いている

    電話がひっきりなしに鳴っている

  3. はず和べき的区别

    はず：语气较为温和，表示预期或逻辑上的必然性。

    べき：语气较为强烈，表示义务、责任或强烈建议。

  4. エッセンス（essence）

    本质、精华

    この本のエッセンスを理解することが重要です

  5. 転がる

    滚动、滚落、摆着

    石が坂道を転がっていく

    その辺には本がたくさん転がっている

阅读1

  1. 遊興性（ゆうきょうせい）

    娱乐性质、消遣倾向

    遊興性のある場所でリラックスしたい

  2. みすぼらしい

    破旧的，寒酸的，邋遢的，不体面的

    彼はみすぼらしい服を着ていた

  3. 台頭する（たいとうする）

    崛起，兴起，显现

    新しい技術が台頭してきている

  4. 見て取る

    看出、察觉、理解

    彼の表情から緊張していることを見て取ることができた

    その場の雰囲気を見て取り、すぐに行動した

  5. 担い手（にないて）

    挑东西的人;中坚;肩负重任的人

    一家の担い手

  6. における和において的区别

    における：表示“在…方面”或“在…的情况下”

    において：表示“在…的场所、领域或情况下”

    教育における問題

    会議において

阅读2

  1. 原発（げんぱつ）

    原子力発電所的缩写

    福島の原発事故は世界的に注目を集めた

  2. 直後（ちょくご）

    正后方，刚…不久

    地震の直後に余震が続いた

  3. 萎縮（いしゅく）

    経済の不況により、多くの企業が萎縮している

  4. ハードディスク

    hard disk,硬盘

  5. 滅多に（めったに）

    很少，不常，几乎不，与否定形式动词搭配

    彼は滅多に映画を見に行かない

  6. 壊滅的（かいめつてき）

    壊滅的な地震が都市を襲った

阅读3

  1. 俗説(ぞくせつ)

    俗说，民间传说

    その病気について多くの俗説がある

  2. くしゃくしゃ

    皱巴巴，揉成一团

    紙をくしゃくしゃにして捨てた

  3. 利口

    聪明，伶俐，机灵

    利口な子供

  4. 驚異的(きょういてき)

    惊人的，可惊的，令人瞠目的

    驚異的な記録

  5. 欠如(けつじょ)

    缺乏，缺少

    責任感が欠如している。

  6. 芸当(げいとう)

    绝技，把戏

    彼女のダンスの芸当はプロ並みだ

阅读4

  1. 躊躇(ちゅうちょ)

    躊躇することなく引き受けた

  2. どんぶり鉢(どんぶりばち)

    丼ものや麺類などを盛り付ける際に使う食器で、プラスチック製や陶器製、木製などがあります

  3. 一年坊主（いちねんぼうず）

    形容那些在某件事情上刚开始时非常热心，但很快就失去兴趣并放弃的人

    新しい趣味を始めても、一年坊主にならないように気をつけている

  4. 多分に甘美で、どこかに秘密にしなければならぬような影を持った物哀しさがあった

    描述某种情感或氛围，特别是带有复杂感情的情景

    过于的甜蜜，带有某种必须隐瞒的阴影的忧伤感

翻译

  1. 呼び寄せる

    召唤、叫来

    鳥を呼び寄せる笛の音

  2. 孕む（はらむ）

    怀有，包含

    危険性をはらんでいます

  3. 図式（ずしき）

    图表、示意图

    この本には多くの図式が含まれています

完型

    1. 気がする：心里感到，好像；总觉得；有心思；动心

      彼女のことが気になる

    3. 気にする：介意，担心、挂念

      失敗を気にするな

    4. 気がつく：意识到，注意到、苏醒，清醒

      間違ったところに気がついた

2020

完型

  1. 願いを抱く

    怀有梦想

  2. 相対的な事柄でさえないのだとしたら

    如果连相对的事情都不是的话

    でさえない：表示“甚至连……都不是”的意思。这里用于强调程度，指连最基本的相对性都不具备。

  3. たちまち

    忽，忽然，突然、不一会儿；转眼间；立刻；眨眼之间

    緊張した空気は、たちまち和らいできた

  4. 実践（じっせん）

  5. ながら用法

    1. 逆接（虽然……但是……）

      彼は忙しいと言いながら、手伝ってくれた

    2. 动作的同时进行（并行）

      音楽を聞きながら勉強する

    3. 原样；原封不动；如……一样

      この種の大根は皮ながら食べたほうがおいしい

    4. 全部，都，表示同类事物处于相同状态中

      兄弟三人ながら人気のある歌舞伎俳優である

    托尔斯泰说，无论人类如何虐待自然，春天依然是春天（即使人类再怎么虐待和破坏自然，春天还是会如期而至）

  2. 石畳（いしだたみ）

  3. 凋落（ちょうらく）

    草木凋落の候

  4. 照応（しょうおう）

  5. 逸脱（いつだつ）

  6. 帯びる（おびる）

阅读3

  1. 生老病死（しょうろうびょうし）

  2. そこでどうするかというと

    “那么，在这种情况下，我们该怎么办呢？”或“那么，接下来该怎么做呢？”这个表达通常用来引出某种行动或解决方案

    生老病死をなぜかみな嫌がります、できれば考えたくない。そこでどうするかというと、例えば人間が生まれるのも特別なことだから、病院へ行ってくれというのです

    那么，接下来该怎么办呢？比如说，人类的诞生是一件特别的事情，所以需要去医院

  3. なりゆき（成り行き）

    动向，趋势；发展，过程；推移，变迁，演变；结果，结局

    成り行き次第で考えなくてはならない
