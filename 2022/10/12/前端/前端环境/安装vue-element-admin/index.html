安装

  • 需要切换分支到 i18n，否则不支持国际化（中文）功能
  • npm install 要多试几次，因为中间会连接 gitbub 下载一些依赖，网络不稳定会导致失败
  • npm run dev 运行后回自动打开浏览器，使用的端口是 9527
  • 安装失败可以试试npm install 安装tui-editor报错解决_Thorold’s Deer的博客-CSDN博客_tui-editor vue
