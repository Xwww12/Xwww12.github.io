混合模式

和当前图层的下面那个图层混合

都是对灰度值（亮度）进行计算，白色为1，黑色为0

  • 变暗：取二者小的
  • 变亮：取二者大的
  • 相加：二者相加
  • 相减：二者相减
  • 相乘（正片叠底）：和纯白相乘为原来的颜色，和纯黑相乘为纯黑，和其他相乘都会变暗（因为灰度在0~1间，怎么乘都会变小）。扣白留黑
  • 屏幕（滤色，screen）：扣黑留白
  • 叠加：亮的地方更亮，暗的地方更暗
  • 差值：取二者之差的绝对值

摄像机跟踪

打开方式

方法一：动画 -> 跟踪摄像机

方法二：窗口 -> 跟踪器 -> 跟踪摄像机

作用

贴图、抠像

使用方法

先shitf选取多个点，右键设置地平面和原点

选择多个点，右键构建实底和摄像机，就可以创建一个纯色图层定在这几个点上

再按住shift选择素材（打开三维开关）的父级关联器，选择纯色图层，就能将素材固定到这个位置（这时纯色图层可以隐藏掉）

运动跟踪

打开方式

方法一：窗口 -> 跟踪器 -> 跟踪运动

作用

和摄像机跟踪的区别在于，运动跟踪只作用于XY轴，适合在平面的跟踪

使用方法

将跟踪点框住要跟踪的目标，点击分析

之后选择编辑目标，选择素材，之后点应用

时间码

打开方式

创建一个纯色图层 -> 效果 -> 文本

其中，编号用来计时，时间码显示的是时间线上的时间

常用表达式

表达式的优先级高于关键帧

打开

按alt在有码表的地方点击即可

按两下e打开有表达式的属性

使用

  • value：这个属性的初始值

  • time：当前时间轴的时间（秒）

  • index：素材的序号

  • wiggle(频率, 振幅)：晃动，返回值的个数会根据属性变化

  • random(最小值, 最大值)：随机数

  • Math.round()：四舍五入取整

  • Math.floor()：向下取整

  • Math.ceil()：向上取整

  • Math.exp(): 指数函数

  • linear(t, tMin, tMax, value1, value2)：将t在[tMin, tMax]的值映射成[value1, value2]

  • if/else

  • inPoint/outPoint：合成的开始时间和结束时间（秒）
