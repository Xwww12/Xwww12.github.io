下载Ollama

Ollama 是一个可以在本地运行大语言模型（LLM）的工具和平台

Ollama官网

步骤：

  1. 下载：进入官网，点击download，选择对应操作系统下载安装包
  2. 安装：进入安装包所在文件夹，进入命令行，通过OllamaSetup.exe /DIR=E:\Ollama指定文件夹安装，否则默认装到C盘
  3. 设置模型的保存路径：新建一个models文件夹，添加环境变量OLLAMA_MODELS，值为这个文件夹。为了保险，启动后右击右下角Ollama图标，选择settings，将Model location路径也改一下
  4. 双击ollama app.exe启动

命令

命令 作用
ollama –version 查看ollama版本
ollama list 列出已经下载到本地的模型
ollama rm <模型名> 删除已下载的模型
ollama help 查看帮助文档
ollama run <模型名> –file <文件名> 从文件输入
ollama run <模型名> –verbose 运行时输出日志
ollama run <模型名> –num-gpu 50 指定显存分配比例

下载模型

步骤：

  1. 查看模型：进入官网，点击Models查看模型，后面的…b为参数的数量(单位十亿)
  2. 下载模型：点击需要的模型，复制右上角，在命令行里运行
  3. 运行模型：下载模型的命令就是运行命令
