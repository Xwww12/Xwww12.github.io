CentOS安装

CentOS8

下载镜像

https://www.centos.org/download/

如要下载CentOS8：

  1. 点击Older Versions下的网址
  2. 点击8.5.2111
  3. 点击isos
  4. 点击CentOS-8.5.2111-x86_64-dvd1.iso即可

安装镜像

安装镜像：

  1. 打开VMware，点击新建虚拟机（用的Vmware Pro 16版本）
  2. 选择“典型”，下一步
  3. 选择“稍后安装操作系统”，下一步
  4. 选择客户机操作系统“Linux”，版本“CentOS 8 64位”， 下一步
  5. 填写虚拟机名称及存放位置，下一步
  6. 最大磁盘大小默认即可，将虚拟磁盘拆分成多个文件，下一步
  7. 自定义硬件
    1. 选择新CD/DVD（IDE）
    2. 使用ISO映像文件，选择下载的iso镜像文件
  8. 点击完成

编辑虚拟机设置：

  1. 不需要声卡、打印机，可将其移除
  2. 网络适配器选NAT
  3. 内存根据需要选择

安装操作系统：

  1. 点击“开启此虚拟机”
  2. 默认第一个“Install CentOS Linux 8”回车
  3. 选择语言“中文”
  4. 分区
    1. 点击“安装目的地”
      1. 存储配置“自定义”，点击“完成”
      2. 点击“+”新增挂载点（大小根据需要改变）
        1. / 17GiB
        2. /boot 1024Mib
        3. swap 2GiB
    2. 点击“完成”，接收更改
    3. 点击”软件选择”，“带GUI的服务器”，选择“开发工具”、”.NET核心开发”、”图形管理工具”
    4. 点击”网络和主机名”，点击”打开“
    5. 点击”根密码”，配置root的密码
    6. 点击”开始安装”
    7. 选择接收许可，结束配置

传输文件到虚拟机

通过FileZilla传输

  1. 检查ssh是否配置

    yum list installed | grep openssh-server

    出现@anaconda等字样说明已安装

    若没有则使用 yum install openssh-server安装

  2. 查看ssh服务状态，要为running

    systemctl status sshd

  3. 使用ifconfig查看虚拟机地址

  4. 在FileZilla中填写虚拟机地址，用户名密码为登录虚拟机的用户名密码，端口为22
