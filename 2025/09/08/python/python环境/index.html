python环境

pip

python的包管理工具

  • 查看pip版本

    pip --version

  • 下载包

    pip install 包名

  • 删除包

    pip uninstall 包名

  • 查看已安装的包

    pip list

conda

conda 是一个开源的软件包管理系统和环境管理软件，用于安装多个版本的软件包及其依赖关系，并在它们之间轻松切换。

安装

anaconda 是一个包含了许多常用库的集合版本，miniconda 是精简版本(只包含conda、pip、zlib、python 以及它们所需的包)

  • miniconda 官网：https://conda.io/miniconda.html
  • anaconda 官网：https://www.anaconda.com/download
  1. 点击安装包安装
  2. 选择安装路径，一路next
  3. 在环境变量中添加变量名：Conda，变量值：Scripts和condabin的文件夹路径
  4. 到Path中添加%Conda%
  5. cmd测试conda --version

卸载

找到文件夹下的Uninstall-Anaconda3.exe运行即可

还有在c盘里有些残留，找教程删干净

命令

  • 查看版本：conda --version
  • 安装指定python版本：conda create -n 环境名 python=3.11.9
  • 切换环境：conda activate 环境名
  • 查看所在的环境：conda info --envs
  • 查看所有环境：conda env list
  • 删除已有环境：conda remove -n 环境名 --all
  • 查看当前环境已安装的库：conda list
  • 退出当前环境：conda deactivate

常用包

版本按需求更改，记得在需要的环境里安装

  • pip install numpy==1.23
  • pip install Pandas==1.2.4
  • pip install Matplotlib==3.5.1

PyTorch

由torch、torchvision、torchaudio三个库组成

安装

conda记得切换环境

  1. 查看GPU的CUDA版本：nvcc -V

  2. 一般torch的cuda版本要 <= CUDA版本

  3. 打开官网https://pytorch.org/get-started/previous-versions/

  4. 在官网ctrl+F搜索pip install torch==1.12.0，找到指定版本的安装命令

  5. 检验是否安装成功
