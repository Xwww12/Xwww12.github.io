二进制枚举

通过二进制数的0/1 表示 选择/变换 对应位置上的字符/数字

例如求字符串的所有子集

若字符串长为3，则可以选择或不选择这个位置上的字符来组成子集，有2 ^ 3种构成方式

可以通过0~2^n的二进制来表示选或者不选

二进制枚举适合n比较小的题，n大的话得通过dfs的剪枝来降低复杂度

格雷码

89. 格雷编码 - 力扣（LeetCode）

在格雷码中，任意两个相邻的代码只有一位二进制数不同，最大码与最小码之间也仅一位不同

转换方式：普通二进制记为Bn-1 Bn-2 … B0，格雷码记为Gn-1 Gn-2 … G0

Gn-1 = Bn-1;

Gi = Bi+1 异或 Bi (0 <= i <= n - 2)
