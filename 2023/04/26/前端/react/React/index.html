语法规则

  1. return 只能返回一个元素，有多个需要用个父元素框起来
  2. 标签必须闭合
  3. 可以在标签文本内、属性内用{ }来使用JavaScript语法
  4. {{ }}表示传递一个对象，在属性style中会用到
